Thesis : Extragalactic galaxies analysis in X-ray

Title : The X-ray AGN population of galaxies in nearby groups

supervisor : Dr.Ewan O’Sullivan (Harvard Smithsonian CfA) & Prof. J. Mimouni (LPMPS, Constantine)

skills : An understanding of X-ray characteristics in galaxies and groups of galaxies, as well as the functioning of
the XMM-Newton and Chandra X-ray observatories. Using TOPCAT for data cross-matching with X-ray telescope catalogs
and the CLoGS sample to isolate active galactic nuclei (AGNs) within the sample. Studies were done on selected galaxies in
multiple wavelengths to determine their characteristics.

Master 1 internship : Cosmological simulations analysis

Title : High-redshift galaxy protoclusters at Cosmic Noon:Simulations vs observations

supervisor : Dr. A. Lebrun (LUTH, OBSPM) & Prof. H. Dole (IAP)

skills : An understanding of SPH simulations such as MACSIS and BAHAMAS, including the structure of data in
the .H5 format. Conducted a data analysis using the YT-project and pyNbody projects to visualize the evolution of the
heaviest structures at z = 0 and their initial structure at z > 1.2.

Master 2 internship : ATLAS Front-end boards analysis

Title : Analysis of the ageing of the Fenics Electronics boards, for the ATLAS upgrade

supervisor : Dr. D. Calvet (CNRS, LPC, ATLAS)

skills : A comprehensive understanding of ATLAS Tile Calorimeter electronics and data pipeline. Data analysis
using Python, MySQL, and JavaScript. A custom Python library to automate the analysis of FENICS boards’ accelerated
aging tests and irradiated boards. Isolation of aging response using data. A full documentation website for the library, which
includes tutorials hosted in github was done using MKdocs.
